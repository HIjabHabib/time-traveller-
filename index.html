About

Time Traveler Layout blends colors, layout and greeting text with the time of day. Changes are smooth and subtle to create a pleasing, readable site from sunrise to starry night.

How it works

JS checks current local hour (or slider setting).
It maps the hour to one of four themes: sunrise, day, sunset, night.
Colors, greeting and scene elements animate between states.
